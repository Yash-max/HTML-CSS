<!DOCTYPE html> for HTML5
<html> for start and end of document
2 parts of html doc :
  * head  contains title and meta data
  * body  contains main display content
2 types of tags :
  * regular
  * empty   without closing tag
Tag can have attributes for additional information
Use only one <h1> tag in html file else SEngine spam it.
Except use-case <article> & <section> are same
In <article> we use <section> to divide related content
In <section> we use <article> to divide different content
<div> tag is grouping data in html file to make changes
  in easily using css
We style inside html also using style tag in head or
style attribute in tag.
ResetSS for resetting the default styling of browser so
that it looks same on all browsers
